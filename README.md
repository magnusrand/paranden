# Hei og velkommen!

Dette er en av de første nettsidene jeg lagde selv! 
Den er laget for å samle tegneserier jeg lagde under navnet «På Randen» da jeg var aktiv i readme – et linjeforeningsmagasin for linjeforeningen Abakus på NTNU.

Siden er veldig enkel og bruker kun HTML og CSS. Materialize-css er bruks som et bootstrap-lignende css-rammeverk for å sette opp stilen på siden.

Siden har ikke blitt oppdatert på mange år og vil ikke bli videre oppdater. Fokuset ligger heller på min nåværende nettside tilgjengelig under [magnusrand.no](https://magnusrand.no). 
Du kan likevel besøke denne siden under [paranden.magnusrand.no](http://paranden.magnusrand.no) hvis du ønsker!
